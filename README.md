# Cunsolo3D Online Print & Quote Portal

Professional, self-serve 3D printing quotes for non-technical customers. Upload a model, see pricing and turnaround instantly, and place an order with confidence. Complex jobs auto-escalate to manual review.

## Product Focus

- Simple, reassuring customer flow (upload → preview → quote → decide)
- Clear pricing + turnaround updates
- Automatic escalation for complex or risky jobs
- Customer dashboard for saved quotes and reorders
- Admin view to review, adjust, and update jobs

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Local Development

```bash
npm install
npm run dev
```

Then open http://localhost:3000.
